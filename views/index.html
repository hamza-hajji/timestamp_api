Timestamp API

How to use

Main URL: https://ts-microservice-api.herokuapp.com/

Example use:

With a date string:

https://ts-microservice-api.herokuapp.com/June%2012%201995

{ "unix": 802915200000, "natural": "June 12, 1995" }

With unix timestamp:

https://ts-microservice-api.herokuapp.com/0

{ "unix": "0", "natural": "January 1, 1970" }
